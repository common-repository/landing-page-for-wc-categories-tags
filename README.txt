== Description ==

= Easily Setup Landing Page =

Landing page for WooCommerce Archive Pages plugin allow you to add Rich text for your WooCommerce archive( Shop, Product Category, Product Tags ) pages. It improve your keyword for SEO. No need to edit your theme or plugin files!

= Features of Landing page for WooCommerce archive( Shop, Product Category, Product Tags ) pages =

* Quick to setup
* Increase SEO
* You can add HTML content and design as per your theme design

== Installation ==

1. Install Landing page for WC Archive Pages by uploading the `landing-page-for-wc-categories-tags` directory to the `/wp-content/plugins/` directory.
2. Activate Landing page for WC Archive Pages through the `Plugins` menu in WordPress.
3. Create Landing page for archive( Shop, Product Category, Product Tags ) pages by going to the `Settings > Landing page for WC Archive Pages` menu.
